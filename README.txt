== Description ==

Poker tournaments and freerolls for your website
 
This plugin creates a list of upcoming poker tournaments (FreeRolls, SnG tournaments, …) at well known poker rooms. Get high-quality content and returning visitors for your website! On top of that you can insert your own affiliate links under individual poker rooms and start making money!

Plugin takes data for a listing of tournaments from the web http://www.pokertip.cz/wp-poker-tournaments-en. These data are collected from official XML feeds poker rooms.
 
Why install WP Poker Tournaments? 
 
- Easy installation. 
- Language: Czech, Russian, English.
- Get high-quality content for your website! Also improves SEO. 
- Get returning visitors. 
- Daily updated list of upcoming FreeRolls and other tournaments. 
- Start making money as an affiliate partner of well known poker rooms. 
 
WP Poker Tournaments is highly adjustable: 
 
- Possibility to excerpt FreeRolls or classic tournaments.
- Possibility to choose the source (certain poker rooms) of excerpted list. 
- Possibility to choose number of excerpted tournaments on the list. 
- Possibility to change description of the columns in the table. 
- Possibility to insert the correct starting date of tournaments (depending on your time zone). 
- Possibility to insert internal or external links. 
- Possibility to insert your own affiliate links under individual poker rooms and start making money!
- Possibility to set the no-follow attribute. 
- Possibility to change the design (box for CSS).

Usage:
 
Shortcode: 
[poker-freerolls] and [poker-tournaments] (on posts or pages)
 
PHP: 
echo do_shortcode(“[poker-freerolls]”); and echo do_shortcode(“[poker-tournaments]”);
 
The plugin is fully functional and free!

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'WP Poker Tournaments'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `wp-pokertournaments.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `wp-pokertournaments.zip`
2. Extract the `wp-pokertournaments` directory to your computer
3. Upload the `wp-pokertournaments` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
